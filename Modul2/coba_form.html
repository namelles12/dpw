Jika anda mengklik tombol "Kirim", data formulir akan di kirim ke halaman bernama "/action_page.php"

Kesimpulan Uji Coba

Berdasarkan serangkaian uji coba yang dilakukan, dapat disimpulkan bahwa tombol reset berfungsi untuk mengembalikan semua isian formulir ke nilai awal (default). Atribut target sangat menentukan di mana hasil pemrosesan akan ditampilkan; nilai _self akan membuka di tab yang sama, sedangkan _blank akan membuka tab baru. Terkait metode pengiriman, penggunaan method="get" akan menampilkan data input secara transparan pada URL browser, sehingga lebih cocok untuk pencarian dan memiliki batasan karakter. Sebaliknya, method="post" mengirimkan data secara tersembunyi di dalam HTTP request body, yang membuatnya jauh lebih aman untuk menangani data sensitif atau data dalam jumlah besar karena tidak terlihat pada alamat URL.
