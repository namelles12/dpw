Kesimpulan Hasil Ujicoba:
Berdasarkan praktikum ini, saya mempelajari bahwa SVG (Scalable Vector Graphics) memungkinkan pembuatan gambar berbasis XML yang tetap tajam meskipun diperbesar. Kita dapat mengatur koordinat (cx, cy), warna (fill), garis pinggir (stroke), hingga tingkat transparansi (opacity) secara langsung di dalam kode HTML[cite: 317, 323]. Hal ini sangat efisien untuk membuat ikon atau grafik sederhana tanpa menggunakan file gambar bitmap.
