Judul Artikel

Ini adalah konten utama dari halaman beranda. Halaman ini menggunakan struktur HTML5 semantik yang terdiri dari header, nav, section, article, dan footer. Layout ini dibuat responsif sehingga tampilannya akan menyesuaikan saat dibuka melalui perangkat mobile atau tablet.

Kesimpulan Hasil Ujicoba:
Melalui latihan ini, saya memahami bahwa penggunaan tag semantik seperti <nav> dan <article> mempermudah browser dalam mengenali struktur konten[cite: 24, 26]. Selain itu, teknik CSS float dan media query sangat penting untuk membuat tata letak halaman yang rapi dan adaptif.
