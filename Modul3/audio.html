Kesimpulan Hasil Ujicoba:
Melalui latihan ini, saya memahami bahwa tag <audio> memungkinkan integrasi media suara secara langsung di web. Penggunaan atribut controls memberikan kendali penuh kepada pengguna untuk memutar, menghentikan, dan mengatur volume. Pastikan penulisan nama file pada src sesuai dengan file asli di folder agar audio dapat dimuat dengan sukses.
